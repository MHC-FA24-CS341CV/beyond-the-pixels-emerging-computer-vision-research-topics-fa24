This project combines VGGFace embeddings with MTCNN preprocessing to test if face alignment improves verification accuracy.

Requirements
- Python 3.6+
- TensorFlow, Keras, MTCNN, NumPy, Matplotlib, scikit-learn

Setup and Run Instructions
1. Download VGGFace Weights: https://www.kaggle.com/datasets/acharyarupak391/vggfaceweights
2. Download dataseet: https://vis-www.cs.umass.edu/lfw/
